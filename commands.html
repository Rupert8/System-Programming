10 базових команд в Git

  • git init – Ініціалізує новий Git-репозиторій.
  • git clone [url] – Клонує репозиторій.
  • git add [файл] – Додає файл до індексу.
  • git commit -m "повідомлення" – Фіксує зміни в репозиторії.
  • git status – Показує стан файлів.
  • git log – Виводить історію комітів.
  • git pull – Витягує зміни з віддаленого репозиторію.
  • git push – Відправляє зміни на віддалений репозиторій.
  • git branch – Виводить список гілок або створює нову.
  • git merge – Об'єднує гілки.
